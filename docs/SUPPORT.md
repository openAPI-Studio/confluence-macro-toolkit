# Customer Support

## How to Get Help

All support for Confluence Macro Toolkit is handled through GitHub Issues.

### Raising an Issue

1. Go to the [Issues page](https://github.com/openAPI-Studio/confluence-macro-toolkit/issues)
2. Click **"New Issue"**
3. Choose the appropriate template (bug report or feature request) if available
4. Fill in the details:
   - **Title** — a short summary of the problem
   - **Description** — what happened, what you expected, and steps to reproduce
   - **Environment** — Confluence version, browser, and any relevant details
5. Submit the issue

### What to Expect

- Issues are triaged and addressed on a best-effort basis.
- Bug fixes will be released when they can — there are no guaranteed SLAs.
- You're welcome to submit a pull request if you'd like to contribute a fix.

### Security Issues

If you discover a security vulnerability, please **do not** open a public issue. Instead, email the maintainer directly (or use GitHub's private vulnerability reporting if enabled on the repo).
